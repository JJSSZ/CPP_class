# 内存管理
一、全局区
    存放全局变量、静态变量、常量
        全局变量 （主函数外定义的变量）。
        静态变量 （通过static修饰的普通变量）。
        常量 （分字符串常量、通过const修饰的常量）。
二、代码区
    存放代码的区域，具有只读性、共享性的特点。

三、栈区
    栈区的数据由编译器自动开辟、管理和释放。
    栈区主要存放局部变量。

四、堆区
    堆区的数据由程序员进行开辟、管理和释放。
    在C++中， 用new开辟堆区的内存并存放数据，用delete释放堆区的内容。
        new的基本句法 a、new int（10）开辟一个存放在堆区的常数10.。b、new int[10]开辟一个存放在堆区的10个元素的数组.。
    
# 引用
一、引用就是起别名
    int a = 10；
    int &b = a；（给变量a起别名，叫b）。
二、注意事项
    1、引用必须初始化。
    2、引用一旦初始化后不可更改。

三、使用引用可以避免使用指针传递形参。
    使用方法：在子函数中使用引用别名接受实参。比如：void fun（int &a，int &b）

# 内联函数
一、当函数定义比较短，函数调用次数多时，可以使用函数内联的方式。
    函数内联是通过在函数定义的前面加上关键字inline实现，比如：inline int f(int a,int b)
                                                                    {
                                                                        比较短的函数体
                                                                    }

二、C++的函数内联是通过C语言的宏定义实现的，内联函数和宏函数实现的效果是一样的
    在c++中，尽量使用内联函数而不用宏函数。

三、注意事项
    1、内联函数在函数嵌套时不可用。
    2、内联函数不适用于函数体多行的函数，一般内联函数的形式：inline int f(int a,int b) {函数体}  //只占一行

# 函数重载
一、在同一个作用域下面的函数名相同的函数，但是函数的参数不同。

二、注意事项
    1、函数返回值不可以作为函数重载的条件。
    2、引用作为重载条件时，加const和不加const可以作为重载的条件。
    3、函数重载碰到默认参数时，容易遇到二义性，要尽量避免函数重载使用默认参数。
